================================================================================
RECOMMENDED: USE BUILDOZER ON LINUX/WSL2
================================================================================

STEP 1: Install WSL2 on Windows
--------------------------------
1. Open PowerShell as Administrator
2. Run: wsl --install
3. Restart computer
4. Open "Ubuntu" from Start Menu
5. Create username and password

STEP 2: Copy Project to WSL
----------------------------
1. In Ubuntu terminal:
   cd ~
   cp -r "/mnt/c/Users/Nazeef Ahmed Kachi/OneDrive/Desktop/QR Attendance Project/qr_attendance_system" ./qr_app
   cd qr_app

STEP 3: Install Build Tools
----------------------------
sudo apt update
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev
pip3 install buildozer cython==0.29.33

STEP 4: Build APK
------------------
buildozer -v android debug

This will:
- Download Android SDK/NDK (first time only)
- Compile Python + Kivy
- Package everything into APK
- Takes 30-60 minutes first time

STEP 5: Get APK
----------------
APK location: ./bin/qrattendance-1.0.0-arm64-v8a-debug.apk

Copy to Windows:
cp ./bin/*.apk "/mnt/c/Users/Nazeef Ahmed Kachi/Desktop/"

STEP 6: Install on Android
----------------------------
1. Transfer APK to phone (Email/USB/Cloud)
2. Enable "Install from Unknown Sources"
3. Tap APK to install
4. Grant permissions (Camera, Internet)

================================================================================
BEFORE BUILDING: IMPORTANT CHANGES NEEDED
================================================================================

1. DEPLOY FLASK BACKEND TO CLOUD
---------------------------------
The mobile app needs a server. Deploy simple_app.py to:
- Heroku (Free tier)
- Railway (Free tier)
- Render (Free tier)
- PythonAnywhere (Free tier)

Example with Render:
1. Go to https://render.com
2. Create "New Web Service"
3. Connect GitHub repo (or upload files)
4. Build command: pip install -r requirements.txt
5. Start command: python simple_app.py
6. Get URL: https://your-app.onrender.com

2. UPDATE API URL IN main.py
-----------------------------
Open main.py, find line 24:
API_BASE_URL = "http://127.0.0.1:5000"

Change to:
API_BASE_URL = "https://your-app.onrender.com"

3. ENABLE CORS ON FLASK
-----------------------
Add to simple_app.py (after line 10):

from flask_cors import CORS
CORS(app)

Install:
pip install flask-cors

================================================================================
TROUBLESHOOTING
================================================================================

Q: Build fails with "Android SDK not found"
A: Buildozer downloads it automatically. Wait for first build.

Q: APK crashes on start
A: Check you updated API_BASE_URL in main.py

Q: "Permission denied" errors
A: Run: chmod +x buildozer.spec

Q: Can't install APK on phone
A: Enable "Install from Unknown Sources" in Android settings

Q: App can't connect to API
A: Make sure Flask backend is deployed and URL is correct
